# Backend Scripts Archive

This directory contains utility scripts that were moved from `backend/scripts/`.

## Files

These scripts are utility/testing scripts that may be run manually but are not part of the main application flow.

## Files Archived

- `comprehensive_test_30_prompts.py` - Comprehensive test script for 30 prompts
- `integration_test_multi_agent.py` - Multi-agent integration test script
- `verify_flow.py` - Flow verification script (note: original version in `archive/legacy/`)

## Note

These scripts can be run manually if needed. They are kept for reference and potential future use.

The main backend entry point is `backend/run.py` which starts the FastAPI server.
